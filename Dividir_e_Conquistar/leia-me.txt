# Exercicio Dividir 

Problema do Segmento de Soma Máxima, resolvido em Python aplicando o paradigma de Divisão e Conquista

## Como usar

1. Instalar primeiramente o [python3](https://www.python.org/), após 
2. Abrir o CMD do windows ou bash no linux.
3. Navegar até o diretório onde está salvo o arquivo programa1.py
4. executar o programa com python programa1.py
```bash
python programa1.py
```
5. Inserir os números da seguinte forma ( numero, numero), por exemplo:
```bash
-16, 20, -10, 12, 27, -6, -4, 8
```
6. Apertar o Enter.
